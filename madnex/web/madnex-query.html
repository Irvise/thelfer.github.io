madnex-query

Thomas Helfer

07/2020

The madnex-query allows querying a MADNEX file from the command line.

Options

Valid options are :

  • --list-behaviours: list all available MFront behaviours
  • --list-material-properties: list all available MFront material properties
  • --list-materials: list all materials in the MFront subgroup
  • --list-models: list all available MFront models
  • --material: specify the material name. may be used to select material properties, behaviours or models not associated with a material

Example of usage

$ madnex-query --list-materials MFront-test1.madnex 
- UO2
- Zircaloy4
