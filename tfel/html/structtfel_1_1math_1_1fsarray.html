Detailed Description

template<unsigned short N, typename T>
struct tfel::math::fsarray< N, T >

a container for finite size array. This is mainly equivalent to the array class of C++11.

Template Parameters
N: size of the array
T: value type
Note
we could not get rid of this class of favor of std::array because two many classes who inherits from tfel::math::fsarry used direct access to the values through the v member.
