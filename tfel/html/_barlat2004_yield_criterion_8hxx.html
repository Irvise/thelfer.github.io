BarlatStressType< StressStensor > tfel::material::computeBarlatStress (const StressStensor &, const BarlatLinearTransformationType< StressStensor > &, const BarlatLinearTransformationType< StressStensor > &, const BarlatExponentType, const BarlatStressType< StressStensor >)
 This function computes the Barlat yield stress. The Barlat yield stress is defined by:

\[ \phi\left(\underline{\bf s}\right) = \sqrt[a]{ \frac{1}{4}\left( \sum_{i=0}^{3} \sum_{j=0}^{3} \left|s'_{i}-s''_{j}\right|^{a} \right) } \]

where \(s'_{i}\) and \(s''_{j}\) are the eigenvalues computed from two tensors \(\underline{\bf s}'\) and \(\underline{\bf s}''\) resulting from a linear transformation of the stress tensor:

\[ \left\{ \begin{aligned} \underline{\bf s}' &= \underline{\underline{L}}' \,\colon\,\underline{\bf s} \\ \underline{\bf s}'' &= \underline{\underline{L}}''\,\colon\,\underline{\bf s} \\ \end{aligned} \right. \]
