◆ getReservedNames()

std::vector<std::string> mfront::NonLinearSystemSolverBase::getReservedNames ( ) const
overridevirtual
Returns
reserved names that ought to be used by on the non linear system solvers:
  • error : current residual. The exact meaning depends on the underlying solver
  • epsilon : stopping criterium value
  • iter : current iteration, automatically declared as a member
  • iterMax : maximum number of iteration
  • converged : boolean stating that convergence is reached

Implements mfront::NonLinearSystemSolver.

Reimplemented in mfront::SecondBroydenSolver.
