Compute the QR decomposition of a matrix.

Parameters
[in,out]m: the matrix to be decomposed
[out]rdiag: diagonal values of the R matrix
[out]beta: euclidian norm of the columns of m The decomposition is done in-place

This function uses householder transformations to compute a qr factorization of the m by n matrix a. that is, one determines an orthogonal matrix q, and an upper trapezoidal matrix r with diagonal elements of nonincreasing magnitude, such that a = q*r. the householder transformation for column k, k = 1,2,...,min(m,n), is of the form: [ I - (1/beta(k))*v*v ] where v has zeros in the first k-1 positions.
