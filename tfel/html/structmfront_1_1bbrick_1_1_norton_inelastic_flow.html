Detailed Description

describe an inelastic flow defined by a Norton law.

The Norton law is defined by the following associated flow rule:

\[ \underline{\dot{\varepsilon}}^{\mathrm{vp}}= A\,\left(\frac{\left<\phi\left(\underline{\sigma}-\displaystyle\sum_{i}\underline{X}_{i}\right)-R\left(p\right)\right>}{K}\right)^{n} \frac{\partial \phi}{\partial \underline{\sigma}} \]

where:

  • \(<.>\) are the Macaulay brackets
  • \(\phi\) is the stress criterion
  • \(\underline{X}_{i}\) is the \(i^{\mathrm{th}}\) back stress
  • \(R\left(p\right)\) is the isotropic hardening rule
  • \(A\) is the Norton coefficient
  • \(K\) is a stress normalisation coefficient
  • \(n\) is the Norton exponent

if an isotropic hardening rule is defined, additional convergence checks are added to check that, if the convergence of the Newton algorithm is reached:

  • if the flow is active (bpl+id is true), the equivalent plastic strain multiplier must be positive. If negative, the convergence flag is set to false and the flow is desactivated (bpl+id is set to false).
  • if the flow is not active (bpl+id is false), the stress criterion must be lower than the elastic limit given by the isotropic hardening rule. If this is not the case, the convergence flag is set to false and the flow is activated (bpl+id is set to true).
Note
if the stress criterion is the same as the flow criterion, the flow is associative.

BehaviourDescription::MaterialProperty mfront::bbrick::NortonInelasticFlow::Ksf
protected

stress threshold factor (optional).

If defined, this property is used to check if the stress prediction does not diverge during the Newton iterations.

In pratice, it means that if the condition \(\sigma_{\mathrm{eq}}-R>K_{sf}\,K\), the Newton step is rejected.
