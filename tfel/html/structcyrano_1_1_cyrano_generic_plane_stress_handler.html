Detailed Description

template<template< tfel::material::ModellingHypothesis::Hypothesis, typename, bool > class Behaviour>
struct cyrano::CyranoGenericPlaneStressHandler< Behaviour >

Generic handler for the plane stress hypothesis.

This allows behaviours written for the generalised plane stress hypothesis to be used in plane stress computations.
