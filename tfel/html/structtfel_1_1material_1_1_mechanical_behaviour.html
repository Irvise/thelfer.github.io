virtual std::pair<bool,NumType> tfel::material::MechanicalBehaviour< btype, H, NumType, use_qt >::computeAPosterioriTimeStepScalingFactor ( const NumType  ) const
pure virtual
Parameters
[in]dtcurrent time step scaling factor
Returns
a pair containing:
  • a boolean syaing if the behaviour integration shall be considered a success or a failure
  • a scaling factor that can be used to:
    • increase the time step if the integration was successfull
    • decrease the time step if the integration failed or if the results were not reliable (time step too large).

Interfaces to behaviour law shall use the hasAPosterioriTimeStepScalingFactor of the traits class MechanicalBehaviourTraits to know if the behaviour is able to give such a time step scaling factor. If not, behaviours may return the NumType(1) value.
